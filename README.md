En enkel bokningsapp byggd med vanilla JavaScript, HTML och CSS.

Dag 1 
Kollade igenom det startmaterial som gavs ut och satte upp projektet lokalt och öppnade det i VSC.

Dag 2 
Installerade Node.js och json-server. Skapade db.json med fyra filmer och testade att servern fungerade via "npm run server". Lade också till package.json med ett startscript så man slipper skriva hela kommandot varje gång.

Dag 3 
Skapade Movie-klassen i JavaScript och Lade till en getter som formaterar strängen för dropdown-menyn.

Dag 4 
Jobbade med platsväljar-logiken. 

Dag 5 
Fixade så att texten längst ner uppdateras när man väljer platser eller byter film. Bytte också ut dollartecknet mot kr.

Dag 6
Testade hela flödet från start — startade servern, öppnade sidan, valde film, klickade runt på platser. Hittade inga buggar.
